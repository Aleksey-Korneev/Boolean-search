Инструкция по запуску:
1. chmod ugo+x index.sh && chmod ugo+x search.sh - гарантирует возможность запуска sh-файла через терминал.
2. ./index.sh arg1 - создание индекса. Необязательный аргумент arg1 интерпретируется как путь до директории, в которой лежат gz-файлы дампа lenta.ru.
3. ./search.sh arg1 - булев поиск через дерево запросов. Если необязательный аргумент arg1 принимает значение "clean", то в директории "temp_dumps" происходит удаление созданных временных файлов.

Описание файлов:
1. index.py:
    - обработка дампа lenta.ru с проверкой корректности пути к документам
    - построение обратного индекса и вспомогательных структур (словарь { слово : id слова}, словарь { id документа : url сайта }, словарь { url сайта : id документа })
    - сохранение данных в json/Simple9
    - проверка сохранности данных после кодирования и декодирования
2. search.py:
    - декодирование архивированных данных
    - реализация потоковой обработки дерева запросов
    - реализация обратной польской нотации
    - реализация булева поиска
3. simple9.py:
    - Кодирование и декодирование информации.

4. main.py:
    - может быть использован для запуска остальных файлов. Принимает следующие возможные аргументы:
    а) "index" - запускает построение индекса с путем до каталога с дампом lenta.ru по умолчанию.
    б) "index" path  - запускает построение индекса с путем до каталога с дампом lenta.ru, равным path.
    в) "search" - запускает булев поиск.
    г) "clean" - удаляет временные файлы из каталога с временными файлами по умолчанию.
    д) "clean" path - удаляет временные файлы из каталога с временными файлами, равного path.
